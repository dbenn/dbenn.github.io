Dynamic invocation of Osamu Ajiki & Ron Baalke's OrbitViewer applet

Rather than having to create the appropriate HTML for the OrbitViewer applet in a static web page, this page dynamically invokes the applet near the end of the current page given input via a form. Moreover, orbital element text can be obtained from a Minor Planet Centre (MPC) web page and the form data populated automatically, an example of which is given below. Object invocation also adds to a popup menu for later retrieval of orbital elements. Browser cookies are used to persist menu items with an expiry of 90 days.

  • Internet Explorer version 6.0 on Windows 98 and Windows 2000;
  • Mozilla version 1.6 on Windows 2000 and version 1.7b on Windows 98. The page is messed up after applet insertion, but scolling the window corrects this.

Note that the popup menu functionality is currently broken.

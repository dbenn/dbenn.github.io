Meade LPI Image Gallery

In January 2004 I purchased a Meade Lunar and Planetary Imager for my Meade LX90 8" Schmidt-Cassegrain telescope. This page is an ongoing record of the images I've taken that I'm happy with. If a parameter is unspecified, this should be taken to mean that that the default value for the given object was used. Unless otherwise stated, The Gimp is used for image processing.
